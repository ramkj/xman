Some context about this project:
This is an open source project which can be used by Python developers or anyone is in the stage of picking up python. The codebase was built while learning python and some of the relaated technologies.

About this project:
This is a simple tool which helps any one who wants to track their personal expenses (household expenses in a simple way). The tool uses Sqlite database for storing data by default.

This software requires the following (dependencies):

- python3 
- Following python modules
	- flask
	- pep8
	- sqlite3
	- jinja2
	- pg8000
	- pytest
	- pycodestyle
